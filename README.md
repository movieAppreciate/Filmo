## 프로젝트 소개

Filmo는 영화·드라마 리뷰에 관심 있는 사용자를 위한 커뮤니티 앱입니다.
영화 정보 제공, 리뷰 작성, 댓글 기능, 그리고 큐레이션 콘텐츠를 제공합니다.
사용자 간의 활발한 소통과 전문적인 리뷰 공유를 통해 콘텐츠 소비 경험을 한층 더 풍부하게 만들어주는 서비스입니다.
